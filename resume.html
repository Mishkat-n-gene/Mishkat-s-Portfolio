Key Projects

1. End-to-End Long-Read Variant Calling & Benchmarking Pipeline on HPC

Bioinformatics / NGS / High-Performance Computing

Designed and implemented a production-ready, containerized whole-genome variant calling pipeline for PacBio HiFi HG002 data on an HPC cluster using SLURM, Nextflow, and Singularity. The workflow included long-read alignment to GRCh38, BAM sorting and indexing, parallel variant calling using Clair3 and DeepVariant, and benchmarking against the GIAB v4.2.1 truth set using hap.py. The project demonstrated applied competency in scalable workflow orchestration, reference genome management, deep learning-based variant detection, and quantitative performance evaluation (Precision, Recall, F1-Score) within a reproducible bioinformatics framework.

2. Vertebrate Genome Project (VGP) Assembly Pipeline

Bioinformatics / Genome Assembly / NGS Workflow Development

Replicated the Vertebrate Genome Project (VGP) assembly pipeline using publicly available sequencing data and the Galaxy platform. Key tasks included long-read assembly generation, iterative polishing, scaffolding, assembly validation, and quality assessment using tools such as Minimap2, Flye, Racon, Medaka, and BUSCO. The workflow produced a high-quality draft assembly for a model organism, with comprehensive metrics generated via QUAST and BUSCO to evaluate completeness and contiguity.

3. Curation and Standardization of Clinically Significant Genetic Variants into GRCh38 VCF

Bioinformatics / NGS

Extracted genomic positions, interpreted HGVS nomenclature, and generated a standards-compliant VCF (v4.2) file representing multiple variant types for downstream analysis. The VCF was further annotated using bamtools with ClinVar, demonstrating applied competency in variant interpretation, genomic coordinates, and bioinformatics standards.

4. Neuro AI – Predictive Stress-Based Disease Modeling

Neuro-Bioinformatics, Mental Health AI

Developed Neuro AI, a clinical research backed predictive tool integrating neurobiology, psychology, and machine learning to assess susceptibility to stress-induced chronic diseases. Engineered a user interface to collect behavioral input and developed an Artificial Neural Network model predicting long-term disease risk, emphasizing a preventive, non-clinical approach.

5. Computational Epidemiology for Pandemic Forecasting

Bioinformatics / AI in Public Health

Explored the role of computational epidemiology in pandemic forecasting and management using existing COVID-19 datasets. Studied how bioinformatics tools (Nextstrain, GISAID) and AI algorithms tracked SARS-CoV-2 spread globally, including mutation tracking and phylogenetic mapping.

6. Finding Similarities and Evolutionary Relationships Through Protein Sequences

Bioinformatics / Molecular Evolution

Analyzed protein sequence similarities and evolutionary relationships across plants, fungi, and animals using FASTA and Jalview for phylogenetic visualization. Gained hands-on experience in sequence alignment, phylogenetic interpretation, and comparative genomics.

7. Molecular Modeling of Axonal Lipid Membrane using CHARMM-GUI

Computational Cell Biology / Structural Bioinformatics

Simulated the lipid membrane composition of a neuronal axon using CHARMM-GUI, constructing a biologically relevant bilayer to study membrane structure and dynamics. Customized lipid types, built the model, and prepared the system for molecular dynamics simulations.

8. Social Media Strategy for Verstees – A Practical Brand Development

Management Sciences

Developed a social media strategy for Verstees, a contemporary fashion/lifestyle brand, applying management principles to branding, digital engagement, and audience growth. Designed content calendars, competitor analysis, influencer strategies, and KPIs to translate theory into practical marketing outcomes.
